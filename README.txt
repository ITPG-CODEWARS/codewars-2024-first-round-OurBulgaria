РЕШЕНИЕ НА ЗАДАЧАТА!!!

Решението на тази задача се осъществява чрез езика за програмиране HTML и CSS,
които се използва за създаването на уеб страници. За целта съм използвал Bootstrap 5,
който изключително много помага за разработката на сайта и всичко в него било то
компоненти, оформление, дизайн и други. Естествено Boostrap 5 поддържа свой css файл,
който съм добавил още в началото на целия код. За разработката на сайта също така съм
използвал технология като Visual Studio Code, чрез която аз въвеждам самият HTML и CSS,
и целият код като така се осъществява ефективно сайта. Единствено в разработката на сайта
няма други css файлове за всяка страница тъй като те зависят от един CSS файл, а именно
на Bootstrap 5. Също така сайтът съдържа търсачка, която обаче не функционира,тъй като
трябва да усвоя как точно се разработват самите търсачки!
Сайтът е адаптивен за всички мобилни устройства!
Сайтът съдържа цели 16 страници! Благодаря!
